# M3-Labs

## Overview

Module 3 consists of three progressive JavaScript labs designed to build comprehensive understanding from fundamental concepts to advanced patterns. Each lab builds upon previous knowledge while introducing new paradigms and techniques essential for modern JavaScript development.

### Learning Progression

- **L1 - JavaScript Fundamentals**: Core concepts including type coercion, functions, objects, and basic syntax
- **L2 - JavaScript Intermediate**: Array methods, object manipulation, and asynchronous programming basics
- **L3 - JavaScript Advanced**: Complex patterns including closures, decorators, classes, and async/await

### Lab Structure

Each lab follows a consistent structure:

- **Progressive difficulty**: Exercises increase in complexity throughout each lab
- **Practical applications**: Real-world scenarios and coding challenges
- **Comprehensive documentation**: JSDoc with examples and expected outputs
- **Automated testing**: Jest test suites for immediate feedback
- **Git workflow**: Branch, commit, and pull request process

### Development Environment

- **Node.js**: Runtime environment for server-side JavaScript execution
- **Jest**: Testing framework for automated validation
- **ES6+**: Modern JavaScript syntax and features throughout
- **Git**: Version control integration for professional workflow

## Lab Directories

### [L1-JavaScript-Fundamentals](./L1-JavaScript-Fundamentals)

Focus on core JavaScript concepts and syntax fundamentals.

**Key Topics:**

- Type coercion and truthy/falsy values
- Function declarations vs expressions vs arrow functions
- Object manipulation and `this` keyword
- Method chaining and array iteration
- Constructor functions vs classes

### [L2-JavaScript-Intermediate](./L2-JavaScript-Intermediate)

Build on fundamentals with intermediate patterns and data manipulation.

**Key Topics:**

- String manipulation and array methods
- Map objects and data processing
- Floating-point precision handling
- Promise-based asynchronous programming
- Object property iteration and validation

### [L3-JavaScript-Advanced](./L3-JavaScript-Advanced)

Master advanced JavaScript patterns and modern development techniques.

**Key Topics:**

- Closures and function factories
- Decorator patterns and function binding
- Class inheritance and prototypes
- Advanced async/await patterns
- Promise chains and error handling

## Getting Started

1. **Fork and clone the forked repository**: `git clone <repository-url>`
2. **Navigate to labs**: `cd M3-Labs`
3. **Choose your lab**: Enter the appropriate directory (L1, L2, or L3)
4. **Follow lab instructions**: Each README.md contains specific setup and completion steps
5. **Run tests**: `npm run test` to validate your solutions
6. **Submit work**: Follow Git workflow for each lab

## Prerequisites

- **JavaScript fundamentals**: Variables, functions, objects, and arrays
- **Basic Git knowledge**: Cloning, branching, and committing
- **Terminal proficiency**: Command-line navigation and execution
- **Problem-solving skills**: Logical thinking and debugging techniques

## Success Criteria

Completion of all three labs demonstrates proficiency in:

- **Modern JavaScript syntax**: ES6+ features and patterns
- **Asynchronous programming**: Promises, async/await, and timing functions
- **Object-oriented programming**: Classes, inheritance, and prototypes
- **Functional programming**: Closures, decorators, and higher-order functions
- **Testing and validation**: Writing robust, testable code
